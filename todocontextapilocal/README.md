# 📝 Todo App

A simple and interactive Todo application built using React.  
This app allows users to manage daily tasks efficiently with features like adding, updating, deleting, and marking tasks as completed.

---

## 🚀 Features

- ➕ Add new tasks  
- ✏️ Update existing tasks  
- ✅ Toggle task completion (line-through effect)  
- 🗑️ Delete tasks  
- ⚡ Instant UI updates
